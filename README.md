
# BitBelt: Blockchain BJJ Lineage & Belt Tracker

## Project Mission
BitBelt is a decentralized verification platform for Brazilian Jiu-Jitsu. It uses Soulbound Tokens (SBTs) on the Base blockchain to create an immutable, verifiable lineage of promotions.

## Tech Stack (The Source of Truth)
- **Frontend:** React Native (Expo)
- **Styling:** Tailwind CSS (via NativeWind)
- **Blockchain:** Thirdweb SDK v5 (Account Abstraction enabled)
- **Network:** Base Mainnet / Base Sepolia (Testnet)
- **Auth:** Social Login (Google/Email) with gasless transactions (sponsored).

## File Naming Conventions
- Components: PascalCase (e.g., BeltCard.tsx)
- Screens: kebab-case (e.g., promotion-screen.tsx)
- Use Expo Router for file-based navigation.
